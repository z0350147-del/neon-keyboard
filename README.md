# Neon Keyboard — Landing Page

## 📝 Project Description
A modern, sleek, and fully responsive landing page dedicated to custom mechanical keyboards. This project was built with a strong focus on a dark neon aesthetic, glowing CSS effects, smooth interactive animations, and flawless adaptation across all mobile and tablet screens.

## 🛠 Tech Stack
- **HTML5:** Semantic layout and clean structure.
- **CSS3:** Custom properties (variables), Flexbox, Grid, keyframe animations, and neon box-shadow effects.
- **GitHub Pages:** Fast and reliable production hosting.
